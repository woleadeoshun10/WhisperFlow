# WhisperFlow

WhisperFlow is a simple Node.js + Express application that fetches a random secret from an external API and displays it using an EJS template. Every page refresh shows a new anonymous secret and the user who submitted it.

---

##  Features
- Fetches random secrets from the Secrets API  
- Displays secret + username  
- Uses EJS templating for dynamic content  
- Styled with custom Google Fonts  
- Simple and beginner-friendly Express setup  

---

##  Installation

### 1. Install dependencies
```bash
npm install

```
### 2. Run the Server
```bash

node index.js

```
### 3. Open in your browser
```bash

https://localhost:3000
```
